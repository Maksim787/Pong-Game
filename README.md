# Pong-Game

## Installation
- Download Release Folder
- Open Pong.exe

## Usage
To start game press **SPACE**
### General
- Pause - **SPACE**
- Restart - **R**
- Exit - **Esc**

### Player1
- Moving UP - **W** or **A**
- Moving DOWN - **S** or **D**

### Player2
- Moving UP - **I** or **L**
- Moving DOWN - **K** or **J**
